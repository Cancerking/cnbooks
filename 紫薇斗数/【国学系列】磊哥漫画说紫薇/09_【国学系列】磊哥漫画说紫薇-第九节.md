### **六亲宫位的两分法原则 [10:13](#?seek_t=613)**
- **六亲与六事宫的区别**  
  六事宫（如财帛、事业）多即少、顺即顺，无需两分；但六亲宫（父母、子女、夫妻、兄弟、朋友）需注意双向互动关系。
- **核心思想：关系是相互的**  
  四化在六亲宫不仅反映对方对自己的影响，也包含自己对该亲的影响。
- **具体分析**  
  - **化禄在父母宫**：我给予父母关爱较多 → 父母回馈也多 → 关系融洽。
  - **化权在父母宫**：父母对我控制欲强 → 我反抗 → 引发纷争。
  - **化科在父母宫**：父母给予关爱多 → 我亦回馈关爱 → 正向循环。
  - **化忌在父母宫**：关心方式不当（如期望过高）→ 导致紧张关系。
- **自我归因的重要性**  
  对子女、兄弟、配偶等关系紧张时，应反思自身是否强势或方法不当，不能全归咎于对方。


### **化禄的核心含义：多与转运 [15:35](#?seek_t=935)**
- **基本定义**  
  化禄的核心含义是“多”，引申为丰盈、顺畅、不争。
- **性格特征推导**  
  - 因资源充足，故不计较得失，性格圆融。
  - 不易与人起纷争，人缘好。
  - 相对懒惰，缺乏奋斗动力。
- **桃花的特殊解读**  
  - 夫妻、子女、交友宫见禄 → 桃花多。
  - 实质是“婚姻多”、“肉体享受多”、“社交机会多”。
  - 并非全然正面，可能导致婚外情、约炮等问题。
  - 从一而终的婚姻更理想，因此此类配置未必吉利。


### **化禄作为转运宫的判断与应用 [19:19](#?seek_t=1159)**
- **转运宫定义**  
  化禄所在宫位即为人生转运的关键领域。
- **贵人的识别方法**  
  - 禄所在宫的地支对应生肖，该生肖之人可视为贵人。
  - 例如禄在寅宫，则属虎者为贵人。
  - 同一生肖也可能带来破财，需辩证看待。
- **人事宫的作用**  
  - 禄在夫妻宫：结婚可转运，婚后财运上升。
  - 禄在子女宫：生子后财运改善。
  - 禄在兄弟/交友宫：多社交、合伙求财有利。
  - 禄在田宅宫：从房产获利。
  - 禄在官禄宫：事业顺利，靠工作赚钱。
  - 禄在命、疾厄、福德宫：皆需靠自己努力。


### **流年穿宫法的应用 [23:51](#?seek_t=1431)**
- **穿宫原理**  
  生年四化可穿透至大运、流年对应宫位，产生实际影响。
- **流年财帛宫来财方向**  
  若原局某宫有生年禄，当流年该宫变为财帛宫时，当年财运佳。
- **迁移宫示例**  
  - 生年禄在申宫，2022年流年迁移宫在申 → 外出发展顺利，人缘好，易获他人帮助。
  - 2023年流年交友宫在申 → 适合合作求财。
- **叠加效应**  
  若生年禄与流年禄同宫，则力量加倍（$L \times 2$）。


### **化禄在十二宫的具体解释 [35:34](#?seek_t=2134)**
- **命宫化禄**  
  - 衣食无忧，性格圆融，人缘好。
  - 懒惰，难成大富，多为富二代特质。
  - 易得外人喜爱，因乐于给予。
- **兄弟宫化禄**  
  - 兄弟朋友关系好。
  - 转运点在兄弟/朋友，适合合作。
  - 容易遇到志同道合者，利于创业。
  - 但也可能出现“合作不长久”现象。
- **夫妻宫化禄**  
  - 异性缘好，感情融洽。
  - 但易出轨，婚姻不稳定。
  - 出轨概率高，男女皆然，女性因母性牵绊稍低。
  - 婚后运势上升，配偶助力大，事业改善。
- **子女宫化禄**  
  - 与子女、学生、晚辈关系融洽。
  - 子女孝顺，晚年有人照顾。
  - 桃花多，易有婚外情。
  - 家庭和睦，因对家人付出多。
- **财帛宫化禄**  
  - 财运佳，衣食无忧。
  - 靠自己赚钱，但因懒惰难成巨富。
  - 婚姻稳定（因有钱不易分手）。
  - 照福德宫 → 有福气。
- **疾厄宫化禄**  
  - 身体享受多，喜吃喝玩乐。
  - 易肥胖，体内易有增生、囊肿等。
  - 多数面临手术风险。
  - 照父母宫 → 父母在外成就高。
- **迁移宫化禄**  
  - 外出发展机遇多，人缘好。
  - 适合向外发展，外地成就大于本地。
  - 即使不出门也不损根本，但外出收益更高。
  - 心情或财运不佳时，外出可转运。
  - 经常旅游或移动也有助运势。
- **交友宫化禄**  
  - 朋友缘分好，人脉广。
  - 成就较高，易成网红、明星、大V。
  - 得陌生人青睐，观众缘好。
  - 合作求财顺利，但需防“先赚后赔”或友情破裂。
- **官禄宫化禄**  
  - 事业顺利，易遇喜欢的工作。
  - 工作机会多，跳槽频繁且顺利。
  - 官禄宫为气数位 → 主长寿、运气好。
  - 照夫妻宫 → 配偶易有外遇。
- **田宅宫化禄**  
  - 房产收益多，家庭物质条件好。
  - 家庭和睦（因经济无忧）。
  - 照子女宫 → 对子女物质支持多。
- **福德宫化禄**  
  - 福气多，知足常乐。
  - 精神世界丰富，兴趣广泛。
  - 钱财想法多，较易生财。
  - 可理解为前世善行果报。
- **父母宫化禄**  
  - 得父母、长辈、领导帮助。
  - 与长辈关系融洽。
  - 适合公务员等体制内职业。
  - 主学业好，身体健康受父母关爱影响。


### **化忌的核心含义：缺与执念 [01:26:44](#?seek_t=5204)**
- **基本定义**  
  化忌核心为“缺”，引发重视与执念，欲弥补空缺。
- **负面属性**  
  属煞星，主不顺、是非、纷争、沟通方式不当。
- **辛苦劳碌**  
  因各方面不顺，只能靠自己努力，导致劳碌。
- **贵人识别反向法则**  
  生年忌所在宫的地支生肖，生活中需特别小心提防。


### **化忌在十二宫的具体解释 [01:33:21](#?seek_t=5601)**
- **命宫化忌**  
  - 自我、自大、固执、自卑。
  - 遇事自我折磨，不愿求助。
  - 利于上班，不利创业（因决策固执易亏损）。
- **兄弟宫化忌**  
  - 与兄弟朋友交流少，感情淡薄。
  - 更愿付出，易因帮助朋友而破财。
  - 建议不借钱给朋友。
  - 亲力亲为，无团队，成就有限，劳碌命。
- **夫妻宫化忌**  
  - 感情欠缺，有执念，急于寻找伴侣。
  - 婚后因期待过高而相处不佳。
  - 离婚率高，或长期互相折磨。
  - 配偶性格亦固执挑剔（因夫妻宫为配偶命宫）。
- **子女宫化忌**  
  - 重视子女，但教育方式不当（如期望过高）。
  - 亲子关系紧张。
  - 桃花方面：无正常桃花或烂桃花多（吃喝嫖赌）。
  - 性生活不美满，可能有功能障碍。
- **财帛宫化忌**  
  - 非贫穷，而是“存不住钱”。
  - 赚得到花得光，钱财流转快。
  - 不花反而易出事（破财消灾）。
  - 重视利润率，对金钱数字极度敏感。
  - 投资不利，不宜借贷。
  - 婚后因财务问题易争吵。
  - 冲福德宫 → 不易开心，快乐短暂。
- **疾厄宫化忌**  
  - 重视身体，注重养生保健。
  - 先天体质较弱（80分 vs 正常100分）。
  - 易因身体不适而情绪烦躁。
  - 冲父母宫 → 脾气不好，与父母关系差。
  - 兄弟宫为疾厄宫气数位，亦反映体质问题。
